# Vue.js 中文社区
---
## 前言
---
项目灵感来源于各种 Vue 版的 cnode 社区，索性我也写一个练练手。
既然是用 Vue 写，那就选择 Vue.js 中文社区了，感谢 Vue.js 中文社区提供的API。

## 技术栈
---
- vue-cli
- vue2.0
- vue-router
- axios
- muse-ui
- marked

## 执行项目
---
> 安装依赖
> npm install

> 启动服务
> npm run dev
